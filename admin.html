ভিডিও বিজ্ঞাপন যোগ করুন

শুধু ভিডিওর লিংক বসাও, নিচে বাটনে ক্লিক করলে কোড নিজে থেকে তৈরি হয়ে যাবে। তারপর সেই কোড কপি করে GitHub-এ ads-config.js ফাইলে পেস্ট করবে।

ভিডিওতে গিয়ে Share → Embed করলে এই লিংক পাবে, অথবা সরাসরি ভিডিওর URL ব্যবহার করো।

পুরো ব্যাপারটা সহজভাবে বোঝো

তোমার অ্যাপে বিজ্ঞাপন দেখানোর জন্য ভিডিওটা কোথাও অনলাইনে থাকতে হবে — তোমার ফোনে সেভ করা ভিডিও সরাসরি অ্যাপে দেখানো যায় না। তাই প্রথমে ভিডিওটা কোনো প্ল্যাটফর্মে (ইউটিউব/ফেসবুক/ইনস্টাগ্রাম/টিকটক) আপলোড করতে হবে, তারপর সেই ভিডিওর লিংক এখানে বসাতে হবে।

নিজের ভিডিও হলে:
১. ভিডিওটা ইউটিউব/ফেসবুকে আপলোড করো (Public বা Unlisted করে দিও, Private করলে দেখা যাবে না)
২. আপলোড হওয়ার পর ভিডিওর লিংকটা কপি করো
৩. সেই লিংক এই ফর্মে বসাও

অন্য কেউ ভিডিও দিলে:
১. যদি সে সরাসরি ভিডিও ফাইল (.mp4) দেয় → সেটা তুমি নিজে ইউটিউবে আপলোড করে সেই লিংক ব্যবহার করো (সবচেয়ে সহজ ও নির্ভরযোগ্য)
২. যদি সে কোনো লিংক দেয় (ইউটিউব/ফেসবুক/ইনস্টাগ্রাম/টিকটক লিংক) → সেই লিংকটাই এই ফর্মে বসিয়ে দাও, নতুন করে আপলোড করার দরকার নেই

কীভাবে আপলোড করবে

১. উপরে কোড কপি করো
২. GitHub রিপোতে ads-config.js ফাইলে যাও
৩. Edit (পেন্সিল আইকন) চাপো
৪. পুরনো const adConfig = {...} অংশটুকু মুছে কপি করা কোড পেস্ট করো
৫. Commit changes চাপো
কয়েক মিনিট পর সব শিক্ষার্থীর কাছে নতুন ভিডিও বিজ্ঞাপন দেখাবে।
